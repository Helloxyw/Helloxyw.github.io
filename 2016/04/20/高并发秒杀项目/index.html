高并发秒杀项目

发表于 2016-04-20

spring-springmvc-Mybatis

SSM整合实现Java高并发秒杀

选择使用springmvc + spring +mybatis三大框架原因是:

  • 轻量
  • 易使用

主要使用到的技术：

Mysql

  • 表设计
  • SQL技巧
  • 事务和行级锁

MyBatis

  • DAO层设计开发
  • MyBatis全程使用
  • MyBatis与Spring整合
  • 推荐XML配置Sql

Spring

  • IOC整合Service
  • 声明式事务运用

Spring MVC

  • Restful接口设计使用
  • 框架运作流程
  • Controller开发技巧

前端

  • 交互设计
  • Bootstrap
  • jQuery
